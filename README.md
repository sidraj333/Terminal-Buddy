# Terminal-Buddy
Coding assistant that runs in the terminal and understands the context of your repository.

## Setup

Use Node 20 with `nvm`:

```bash
nvm install 20
nvm use 20
```

Install dependencies:

```bash
npm install
```

Build the app:

```bash
npm run build
```

## Run The App

After building:

```bash
bdy
```

If `bdy` is not on your `PATH`, run it directly from the repo root:

```bash
./bdy
```
